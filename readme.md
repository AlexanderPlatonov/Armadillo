# Support Subcase Visualization

## How to run development environment
* Install .NET Core 2.1 SDK (Blazor doesn't support 2.2), https://dotnet.microsoft.com/download/dotnet-core/2.1
* `cd Armadillo.Server`
* `dotnet run`
